## 一、游戏概述

昆明双扣属于“升级类（拖拉机体系）”四人对抗牌类游戏，是云南地区最主流的玩法之一。

### 1.1 基本信息
- **使用牌具**：两副扑克牌，共108张
  - 包含大小王：大王2张，小王2张
  - 四门花色：黑桃、红桃、梅花、方块
  - 每门花色有2份从2到A的牌（共13×2=26张）
  - 总牌数：4花色×26张 + 4张王 = 108张
- **人数与组队**：4人，分为两队，**对家为同盟**，两人配合对抗另外两人
- **手牌与底牌**：每人手牌25张，底牌8张
- **总分**：200分（所有分牌分值总和）
- **胜利目标**：通过守分与抢分升级，先打过A的一方获胜
- **核心原则**：必须跟花色，不可随意破坏花色结构

### 1.2 游戏流程概览
```
发牌（每人25张，留8张底牌）→ 亮主/反主（确定庄家、主牌花色、级牌）→ 扣底（庄家换8张牌）→ 出牌（多轮进行，每轮结算分数）→ 最后一轮（抠底/保底）→ 结算（计算闲家总得分）→ 升级/换庄（按得分确定下一局）
```

## 二、牌具与牌型结构

### 2.1 牌的分类
| 分类     | 定义               | 包含                              |
| -------- | ------------------ | --------------------------------- |
| **主牌** | 当前局中最大的牌型 | 常主 + 级牌 + 主牌花色所有牌      |
| **副牌** | 非主牌花色的牌     | 其余三门花色的2-A（不含当前级牌） |
| **分牌** | 计分牌             | 5（5分）、10（10分）、K（10分）   |

| 牌       | 每张分值 | 两副牌张数       | 总分值    |

| 10       | 10分     | 8张（每花色2张） | 80分      |
| K        | 10分     | 8张（每花色2张） | 80分      |
| **总计** |          |                  | **200分** |

| 牌型       | 定义                   | 张数要求 | 示例                 |
| ---------- | ---------------------- | -------- | -------------------- |

## 三、常主、级牌与主牌构成

### 3.1 常主（固定主牌）
以下牌张**始终为主牌**，不受主牌花色变化影响，在任何情况下都属于主牌：
- **大王**：2张
- **小王**：2张
- **黑桃A**：2张（注意：只有黑桃A是常主，其他花色的A不是常主）

### 3.2 级牌（机动主）
级牌是随着升级进度变化的机动主牌，每个级牌在四门花色中各有一对（共8张）。

**升级顺序**（循环）：
> **2 → 5 → 10 → J → Q → K → A → 2**

**必打级牌**：**2、J、A** 为必打级牌，任何升级情况均不能跳过。

### 3.3 有主状态下的主牌构成
当有主牌花色时，主牌由以下三部分构成：

| 组成部分           | 包含内容              | 数量说明                   |
| ------------------ | --------------------- | -------------------------- |
| **常主**           | 大王、小王、黑桃A     | 6张                        |
| **当前级牌**       | 四个花色的当前级牌    | 8张                        |
| **主牌花色全部牌** | 主牌花色的2-A（两份） | 24张（去重后实际计入略少） |

**示例**（打5，亮红桃）：
- 常主：大王2、小王2、黑桃A2 = 6张
- 级牌：红桃5（主5）2张、其他花色5（副5）6张 = 8张
- 主花色牌：红桃2、3、4、6、7、8、9、10、J、Q、K、A各2张 = 24张（红桃5已计入级牌，实际总主牌需去重，约36张）

### 3.4 无主状态下的主牌构成
当打“无主”时，主牌只包含**常主**和**当前级牌**，没有主牌花色：

| 组成部分     | 包含内容           | 数量     |
| ------------ | ------------------ | -------- |
| **常主**     | 大王、小王、黑桃A  | 6张      |
| **当前级牌** | 四个花色的当前级牌 | 8张      |
| **总计**     |                    | **14张** |

**特点**：
- 没有主牌花色，所有副牌花色平等
- 主牌数量少，副牌地位提升

## 五、拖拉机（连对）构成规则

拖拉机是昆明双扣的核心牌型，由**点数相连**且**花色相同**的**至少两对牌**组成。

### 5.1 拖拉机的基本定义
- 必须是对子（两张相同点数的牌）
- 必须点数相连（在主牌顺序或副牌顺序中相邻）
- 必须花色相同
- 至少两连对（4张牌）

### 5.2 副牌拖拉机构成
副牌拖拉机必须**跳过当前级牌**（因为级牌不在副牌中）。

**示例**（以打5为例）：
- **3344**：3和4相邻，构成拖拉机（二连对）
- **7788**：7和8相邻，构成拖拉机
- **99 10 10 JJ**：9、10、J相邻（跳过5），构成拖拉机（三连对）
- **QQKKAA**：Q、K、A相邻，构成拖拉机（三连对）

**不构成拖拉机的情况**：
- **4477**：4和7不相邻，不是拖拉机
- **3355**：5是级牌不在副牌中，所以3355实际上是33和？——这种情况在副牌中不可能出现

### 5.3 主牌拖拉机构成（有主状态）
主牌拖拉机允许**跨类别连接**，遵循主牌的大小顺序。

**主牌大小顺序**（以打5为例）：
> 大王 > 小王 > 主5 > 黑桃A > 副5 > 主花色A > 主花色K > 主花色Q > ……

**拖拉机连接规则**：
- 只要在主牌大小顺序中相邻的两对牌，即可构成拖拉机
- 可以跨常主、级牌、主花色牌

**示例**（以打5为例）：
- **一对小王 + 一对主5**：小王与主5在主牌顺序中相邻，构成拖拉机
- **一对主5 + 一对副5**：主5与副5相邻，构成拖拉机
- **一对副5 + 一对主花色A**：副5与主花色A相邻，构成拖拉机
- **一对主花色A + 一对主花色K**：主花色A与主花色K相邻，构成拖拉机
- **一对大王 + 一对小王**：大王与小王相邻，构成拖拉机

### 5.4 主牌拖拉机构成（无主状态）
打“无主”时，主牌只有常主和当前级牌，顺序为：
> 大王 > 小王 > 当前级牌（所有花色平等） > 黑桃A

**拖拉机示例**：
- **一对小王 + 一对当前级牌**：小王与级牌相邻，构成拖拉机
- **一对当前级牌 + 一对黑桃A**：级牌与黑桃A相邻，构成拖拉机
- **一对大王 + 一对小王**：构成拖拉机

**注意**：无主时，所有花色的级牌大小相同，所以一对级牌（无论什么花色）都算作同一级别的对子。

### 5.5 拖拉机长度与命名
| 连对数量    | 名称             | 张数  | 示例           |
| ----------- | ---------------- | ----- | -------------- |
| 2连对       | 二连对（拖拉机） | 4张   | 3344           |
| 3连对       | 三连对（火车）   | 6张   | 556677         |
| 4连对       | 四连对           | 8张   | 66778899       |
| 5连对       | 五连对           | 10张  | 7788991010JJ   |
| 6连对及以上 | 长拖拉机         | 12张+ | 88991010JJQQKK |

拖拉机越长，威力越大，尤其在抠底时翻倍倍数越高。

## 六、亮主与反主规则

亮主与反主阶段决定三个关键要素：庄家是谁、主牌花色是什么、当前级牌是什么。

### 6.1 亮主时机
亮主发生在**摸牌过程中**。玩家在摸到牌后，可以立即决定是否亮主。

### 6.2 首局抢庄
- 摸牌过程中，**最先亮出当前级牌**（亮单张即可）的人成为庄家
- 亮出的那张牌的花色即成为本局**主牌花色**
- 如果亮出的是一对级牌，则为“**定主**”（见6.4）

**示例**：第一局级牌为2，玩家A摸到红桃2后立即亮出，则玩家A成为庄家，红桃为本局主牌花色。

### 6.3 暗亮主
- 如果玩家摸到的第一张牌就是当前级牌，可以选择**暗亮**（不展示给其他人看，只自己知道）
- 暗亮后，该花色暂时成为主牌花色，但其他玩家仍可反主
- 暗亮者在反主阶段开始前可以保持匿名
- 暗亮者可以在后续亮出同一花色级牌来确认（或被人反主）

### 6.4 定主规则
- 如果玩家亮出的不是单张级牌，而是**一对级牌**（如两张红桃2），则触发“定主”
- **定主后**：该花色被确定为最终主牌花色，此后**不能再反花色**
- 定主后仍然可以反主，但只能**用一对王反为“无主”**

### 6.5 反主规则（完整版）

| 反主类型             | 条件                                   | 结果                       | 备注                         |
| -------------------- | -------------------------------------- | -------------------------- | ---------------------------- |
| **反花色**           | 在“定主”发生前，有人亮出**另一对级牌** | 主牌花色改为新亮出的花色   | 先亮者有效，后亮者覆盖前亮者 |
| **反无主（定主后）** | 在“定主”发生后，有人亮出**一对王**     | 本局变为“无主”             | 只能用王反，不能反花色       |
| **反无主（未定主）** | 在“定主”发生前，有人亮出一对王         | 本局变为“无主”             | 可以跳过花色直接变无主       |
| **反暗主**           | 最后一张牌是本局级牌                   | 可“反暗主”，该花色即为主牌 | 最后一张牌时触发             |

**重要原则**：
- **自己不能反自己**所亮的主（不能自己否定自己）
- 反主可以在**扣底前**进行，也可以在**扣底后**进行（扣底后反主触发“翻底”）
- 反主时，亮出的对子不分花色大小顺序，先亮者优先
- 反主可以多次进行，直到没有人再反为止

### 6.6 反主顺序示例
1. 玩家A亮出红桃5（单张）→ 庄家A，主牌红桃
2. 玩家B亮出一对梅花5 → 反主成功，主牌改为梅花，玩家B成为新庄家
3. 玩家C亮出一对小王 → 反无主成功，本局变为无主，庄家仍为玩家B
4. 玩家D不能再反，因为无主状态后只能反无主，但已经无主了

### 6.7 无人亮主的处理
若发牌结束（所有人25张牌已摸完）仍无人亮主：
- 以**底牌的第一张牌的花色**作为本局主牌花色
- 若第一张牌是大小王，则本局为“**无主**”

## 七、扣底与翻底规则

### 7.1 扣底流程
- 庄家确定后，庄家拿起8张底牌
- 庄家从自己手牌中选出8张牌，**重新扣回桌面**作为新的底牌
- **昆明主流允许扣分牌**（5/10/K可以放入底牌），但要承担被抠底翻倍的风险
- 扣底时不能查看其他玩家的手牌
- 扣底后，庄家手牌应仍为25张（原25张-8张+8张=25张）

### 7.2 扣底策略考虑
- 扣分牌：如果庄家方实力强，有信心保底，可以扣分牌增加闲家得分难度
- 扣主牌：如果庄家方主牌多，可以扣副牌，保留主牌实力
- 扣闲牌：如果某门花色特别多，可以扣掉该花色的小牌

### 7.3 翻底规则（扣底后反主）
如果有人在**庄家扣完底牌之后**才进行反主，则触发“翻底”：

**翻底流程**：
1. 反主者亮出对子（级牌或王）完成反主
2. 反主者有权**查看底牌**
3. 若底牌中有反主花色的牌：
   - 反主者可以**捡起这些牌**，加入自己手牌
   - 反主者必须从自己手牌中选出**相同数量**的牌，**重新扣回底牌**
   - **严禁将分牌替换入底牌**（即替换的牌不能是5/10/K）
4. 若底牌中没有反主花色的牌：
   - 反主者从庄家手中**抽取一张最大的主牌**
   - 庄家从底牌中随机抽一张还给反主者（作为补偿）

**翻底示例**：
- 庄家扣底后，玩家B用一对梅花5反主
- 玩家B查看底牌，发现有2张梅花
- 玩家B捡起这2张梅花，从自己手牌中选出2张非分牌扣回底牌
- 主牌花色改为梅花，玩家B成为新庄家

## 八、出牌核心规则

### 8.1 基本规则
- **出牌顺序**：逆时针方向
- **首出**：每局第一轮由庄家先出牌，之后每轮由前一轮的**获胜方**先出牌
- **出牌落地为准**：牌打出后不得更换

### 8.2 跟牌原则（核心：必须跟花色）
当首家出某一种花色的牌时，其余三家必须遵守以下规则：

| 首家出牌牌型 | 其余三家必须跟出的牌 | 若没有相应牌型                       |
| :----------- | :------------------- | :----------------------------------- |
| **单张**     | 一张同花色的单张     | 可垫牌或用主牌“毙”                   |
| **对子**     | 同花色的对子         | 跟任意两张牌（垫牌）或用主牌对子“毙” |
| **拖拉机**   | 同花色的拖拉机       | 跟对子或单张，或用主牌拖拉机“毙”     |

### 8.3 详细跟牌规定（分情况说明）

**情况一：首家出单张**
- **如果你有该花色**：必须出一张该花色的牌（可以比首家小）
- **如果你没有该花色**：可以：
  - **垫牌**：出任意一张其他花色（副牌或主牌都行，但算作垫牌，不参与竞争）
  - **毙牌**：出主牌（需符合毙牌规则）

**情况二：首家出对子**
- **如果你有该花色的对子**：必须跟出对子（哪怕比首家小）
- **如果你有该花色但只有单张没有对子**：可以出两张单张（垫牌）
- **如果你没有该花色**：可以：
  - **垫牌**：出任意两张其他花色
  - **毙牌**：出主牌对子（需符合毙牌规则）

**情况三：首家出拖拉机（如3344）**
- **如果你有该花色的拖拉机**：必须跟出拖拉机（长度需≥首家拖拉机）
- **如果你有该花色但只有对子没有拖拉机**：可以跟出对子（数量需等于首家拖拉机中的对数）
- **如果你有该花色但对子数量不够**：可以跟对子+单张
- **如果你没有该花色**：可以：
  - **垫牌**：出任意张数相同的其他花色牌
  - **毙牌**：出主牌拖拉机（需符合毙牌规则）

### 8.4 获胜判定
- 每轮四人出牌完毕后，比较谁出的牌最大
- **同花色比较**：按牌力大小顺序（A>K>Q>...）
- **不同花色比较**：主牌 > 副牌（毙牌成功时）
- **同为主牌**：按主牌力顺序比较
- 牌最大的一方获得该轮的所有分牌，并获得下一轮的首出权

### 8.5 牌型组合的处理原则
- 当一家出的牌包含多种牌型（如拖拉机+对子+单张）时，分别比较各部分的牌型
- 只有所有部分都大于对方，才算获胜
- 这种复杂情况主要出现在甩牌中

## 九、毙牌与盖毙规则（详细版）

毙牌是当玩家没有首家花色时，用主牌争夺胜利的重要手段。

### 9.1 毙牌的基本条件
当玩家**没有**首家所出的副牌花色时，可以用主牌来管住这一轮，称为“毙牌”。

**毙牌必须满足**：
1. 玩家手中确实没有首家花色
2. 玩家使用**主牌**出牌
3. 所出的主牌，其**牌型结构**必须**完全匹配**首家所出的牌型结构

### 9.2 毙牌的牌型匹配要求（详细表格）

| 首家出牌类型        | 首家出牌示例         | 毙牌时必须出的主牌                |
| ------------------- | -------------------- | --------------------------------- |
| **单张1张**         | 红桃A                | 主牌单张1张                       |
| **对子2张**         | KK                   | 主牌对子2张                       |
| **二连对4张**       | 3344                 | 主牌二连对至少4张（必须是拖拉机） |
| **三连对6张**       | 556677               | 主牌三连对至少6张（必须是拖拉机） |
| **四连对8张**       | 66778899             | 主牌四连对至少8张（必须是拖拉机） |
| **对子+单张组合**   | AAK（2对+1单）       | 主牌2对+1单（对子数量需匹配）     |
| **拖拉机+对子组合** | 556677+KK（3对+1对） | 主牌拖拉机+对子（数量需匹配）     |

### 9.3 数量匹配的详细规则
毙牌时所出主牌必须满足：
1. **总张数相等**：毙牌的张数必须等于首家出牌的张数
2. **拖拉机数量≥**：毙牌中拖拉机的数量（以“连对”为单位）不得少于首家出牌中拖拉机的数量
3. **对子数量≥**：毙牌中对子的数量（不含拖拉机中的对子）不得少于首家出牌中对子的数量（不含拖拉机中的对子）
4. **单张数量相应**：剩余单张数量自动匹配

### 9.4 毙牌示例大全

**示例1**：首家出副牌单张红桃A
- 毙牌要求：出1张主牌单张
- 可以：出大王、小王、主5等任何主牌单张 ✓
- 不可以：出对子（张数不对） ✗

**示例2**：首家出副牌对子KK
- 毙牌要求：出主牌对子
- 可以：出主5对子、副5对子、主花色A对子等 ✓
- 不可以：出主5+小王（不是对子） ✗

**示例3**：首家出副牌拖拉机556677（6张，含3个对子）
- 毙牌要求：出主牌拖拉机6张（含3个对子）
- 可以：出主牌拖拉机998877（6张，3个对子） ✓
- 可以：出主牌拖拉机99887766（8张，4个对子，数量可多但拖拉机数需够） ✓
- 不可以：出主牌对子AAKK+两张单牌（共6张，但没有拖拉机） ✗
- 不可以：出主牌拖拉机9988（4张，张数不够） ✗

**示例4**：首家出副牌组合AAKK+Q（2对+1单，共5张）
- 毙牌要求：出主牌2对+1单（共5张）
- 可以：出主5对+副5对+单张小王 ✓
- 不可以：出主牌拖拉机（虽有对子但结构不对） ✗

### 9.5 盖毙规则
当一家用主牌毙牌后，后面的玩家如果也没有首家花色，可以用**更大的主牌**且**牌型结构合法**的牌来管住，称为“盖毙”。

**盖毙的条件**：
1. 后面玩家也没有首家所出的副牌花色
2. 后面玩家使用主牌出牌
3. 所出的主牌必须**大于**前面毙牌玩家的主牌（按主牌力顺序比较）
4. 所出的主牌必须**符合毙牌的牌型匹配要求**

**盖毙示例**（以打5为例）：
- 首家出副牌拖拉机556677
- 玩家A（无红桃）用主牌拖拉机998877毙牌
- 玩家B（也无红桃）手中有主牌拖拉机QQJJ1010（比998877大）
- 玩家B可以用QQJJ1010**盖毙** ✓

**不可盖毙的情况**：
- 首家出副牌拖拉机556677
- 玩家A用主牌拖拉机998877毙牌
- 玩家B手中没有拖拉机，只有主牌对子AAKK+两张单牌
- 玩家B不能用AAKK+单牌盖毙（因为没有拖拉机，只能垫牌）

### 9.6 毙牌与垫牌的区别
- **垫牌**：出任何牌（包括主牌），但算作放弃这一轮，不参与竞争
- **毙牌**：出主牌并符合牌型要求，参与竞争，有可能赢得这一轮

**关键区别**：即使出了主牌，如果不符合毙牌的牌型要求，也视为垫牌，不能赢得这一轮。

## 十、甩牌规则

甩牌是当玩家确信自己手中的某门花色的若干张牌均为当前最大时，可以一次性全部甩出。

### 10.1 甩牌条件
- 所有甩出的牌必须是**同一花色**（主牌或副牌）
- 甩出的牌可以包含单张、对子、拖拉机的任意组合
- 必须保证**每一张牌**都是当前该花色中没有人能大过的
- 必须考虑**已出过的牌**和**其他家手中可能还有的牌**

### 10.2 如何判断“当前最大”
判断某张牌是否为当前最大，需要：
1. 记住已经出过的所有牌
2. 推算其他家手中可能还剩什么牌
3. 确保没有其他玩家能大过你的任何一张牌

**判断示例**：
- 你想甩副牌AAK，必须确保：
  - 所有A都还在你手中，其他家没有A了（或者有A但已出完）
  - 所有K都还在你手中，其他家没有K了
  - 同时要确保你想甩的对A是最大的对子（没有更大的对子）

### 10.3 甩牌的跟牌规则
当有人甩牌时，其余三家必须按以下顺序和规则跟牌：

1. **先看拖拉机**：甩牌中有拖拉机，其余家若有同花色拖拉机必须跟出（拖拉机长度≥甩牌中的拖拉机）
2. **再看对子**：甩牌中有对子（不含拖拉机中的对子），其余家若有同花色对子必须跟出（对子数量≥甩牌中的对子数）
3. **最后看单张**：甩牌中有单张，其余家若有同花色单张必须跟出

**跟牌示例**：
- 玩家A甩出主牌：一对大王 + 一对小王 + 主5（拖拉机+对子+单张）
- 玩家B若有主牌，必须：
  - 如果有拖拉机，必须跟拖拉机（长度≥一对大王+一对小王的长度？实际需具体分析）
  - 如果有对子，必须跟对子
  - 如果有单张，必须跟单张

### 10.4 甩牌失败的判定
如果玩家判断错误，甩出的牌中有一张不是当前最大，则构成“甩牌失败”。

**甩牌失败的处理**：
1. 违规者必须从甩出的牌中，打出**最小的那一张牌**（或最小牌型中的一张）
2. 其余牌收回手牌
3. **每错一张牌，罚10分**（罚分计入对方得分或从本方总分扣除，赛前约定）

**甩牌失败示例**：
- 玩家甩出AKQ，但K不是最大（其他家还有一张K）
- 玩家必须改出A（最小的一张），收回K和Q
- 罚10分

### 10.5 甩牌抠底的特殊处理
如果最后一轮是甩牌且闲家获胜（抠底）：
- 若甩牌中**没有对子**（全是单张），抠底按单张计算（×2倍）
- 若甩牌中**有对子或拖拉机**，抠底按拖拉机计算（×8倍起，具体倍数按甩牌中最大牌型计算）

## 十一、计分规则

### 11.1 得分时机
每轮出牌结束后，牌最大的一方获得该轮的所有分牌：
- **庄家方赢墩**：该轮所得分牌**作废**，不计入闲家得分
- **闲家方赢墩**：该轮所得分牌**累加到闲家总分**

### 11.2 分牌分值表
| 牌       | 每张分值 | 两副牌张数 | 总分值    |
| -------- | -------- | ---------- | --------- |
| 5        | 5分      | 8张        | 40分      |
| 10       | 10分     | 8张        | 80分      |
| K        | 10分     | 8张        | 80分      |
| **总计** |          |            | **200分** |

### 11.3 抠底翻倍（核心规则）
最后一轮出牌，若闲家牌最大（抠底），则**底牌中的分牌翻倍**后计入闲家总分。

| 抠底牌型         | 底牌分翻倍倍数   | 示例                           |
| ---------------- | ---------------- | ------------------------------ |
| **单张**         | ×2倍             | 最后一张是单张                 |
| **对子**         | ×4倍             | 最后两张是对子                 |
| **二连对**       | ×8倍             | 最后四张是拖拉机（如3344）     |
| **三连对**       | ×16倍            | 最后六张是拖拉机（如556677）   |
| **四连对**       | ×32倍            | 最后八张是拖拉机（如66778899） |
| **五连对**       | ×64倍            | 最后十张是拖拉机               |
| **六连对及以上** | ×128倍，依此类推 | 拖拉机越长，倍数越高           |

**抠底计算示例**：
- 底牌中有分牌：5（5分）、10（10分）、K（10分），共25分
- 闲家最后一轮用对子抠底（×4倍）
- 闲家获得：25分 × 4 = 100分

### 11.4 保底
若最后一轮庄家牌最大（保底）：
- 底牌中的分数归庄家方所有
- 不计入闲家得分
- 庄家成功保护了底牌

### 11.5 闲家总得分计算
闲家总得分 = 出牌阶段累计得分 + 抠底翻倍得分

**最终得分用于决定升级结果**。

## 十二、升级与轮庄规则

### 12.1 升级基础表
根据闲家总得分，决定升级结果：

| 闲家得分      | 结果                     | 俗称     |
| ------------- | ------------------------ | -------- |
| **0分**       | **庄家方升3级**          | 大光     |
| **5-35分**    | **庄家方升2级**          | 小光     |
| **40-75分**   | **庄家方升1级**          | 正常升级 |
| **80-115分**  | **闲家上台**（级牌不变） | 上台     |
| **120-155分** | **闲家上台且升1级**      | 上台+1   |
| **160-195分** | **闲家上台且升2级**      | 上台+2   |
| **200分**     | **闲家上台且升3级**      | 上台+3   |

### 12.2 升级线计算规则（详细版）
- 以80分为基准线，闲家得分≥80分则闲家上台
- 80分以上，**每20分一级**：

| 得分区间  | 闲家升级级数 | 庄家方变化 |
| --------- | ------------ | ---------- |
| 80-99分   | 上台（0级）  | 下庄       |
| 100-119分 | 上台+1级     | 下庄       |
| 120-139分 | 上台+2级     | 下庄       |
| 140-159分 | 上台+3级     | 下庄       |
| 160-179分 | 上台+4级     | 下庄       |
| 180-199分 | 上台+5级     | 下庄       |
| 200分     | 上台+6级     | 下庄       |

### 12.3 升级示例
**示例1**：庄家打5，闲家得65分
- 65分在40-75分区间
- 庄家方升1级，下一局打10
- 庄家继续坐庄（或由对家坐庄，见轮庄规则）

**示例2**：庄家打10，闲家得95分
- 95分≥80分，闲家上台
- 闲家得分在80-99分区间，上台不升级
- 下一局新庄家从10开始打起

**示例3**：庄家打J，闲家得145分
- 145分≥80分，闲家上台
- 145分在140-159分区间，上台+3级
- J→Q→K→A，升3级到A
- 下一局新庄家从A开始打起（但需考虑必打牌限制）

### 12.4 轮庄规则
- **庄家升级时**（闲家未上台）：下一局由本局庄家的**对家**当庄家（庄家方内部轮换）
- **闲家上台时**：下一局由本局庄家的**下家**当庄家（新庄家方开始坐庄）

**轮庄示例**：
- 座位：北（庄家）、东、南（对家）、西
- 庄家升级：下一局南（对家）当庄家
- 闲家上台：下一局东（下家）当庄家

## 十三、J/A特殊降级规则（完整版）

当级牌为J或A时，除了正常升级规则外，还有特殊的降级风险。**必须同时满足以下三个条件**才会触发降级。

### 13.1 打J时的降级规则
| 项目         | 内容                                                                                                                |
| ------------ | ------------------------------------------------------------------------------------------------------------------- |
| **触发条件** | ① 当前级牌为**J**（庄家正在打J）<br>② 闲家总得分 **≥80分**<br>③ **最后一轮**由闲家出**一对J**且这轮牌为最大（抠底） |
| **结果**     | 庄家由J **降级到2** 并下庄                                                                                          |
| **下一局**   | 新庄家从**2**开始打起                                                                                               |

### 13.2 打A时的降级规则
| 项目         | 内容                                                                                                                |
| ------------ | ------------------------------------------------------------------------------------------------------------------- |
| **触发条件** | ① 当前级牌为**A**（庄家正在打A）<br>② 闲家总得分 **≥80分**<br>③ **最后一轮**由闲家出**一对A**且这轮牌为最大（抠底） |
| **结果**     | 庄家由A **降级到J** 并下庄                                                                                          |
| **下一局**   | 新庄家从**J**开始打起                                                                                               |

### 13.3 保级机制——“落实政策”
如果闲家得分 **<40分**，则即使闲家在最后一轮用一对J/A抠底，庄家也**不降级**，下一局继续打J/A。

**“落实政策”的含义**：
- 闲家得分太低（小光以内），没有资格让庄家降级
- 庄家获得保护，继续坐庄打同一级牌

### 13.4 场景示例大全

**场景一**：打J，闲家得分85分，最后一轮用一对J抠底
- 条件① ✓ 级牌是J
- 条件② ✓ 85分 ≥ 80分
- 条件③ ✓ 最后一轮用一对J抠底
- **结果**：庄家降级到2并下庄

**场景二**：打J，闲家得分85分，最后一轮用一对10抠底
- 条件① ✓ 级牌是J
- 条件② ✓ 85分 ≥ 80分
- 条件③ ✗ 最后一轮不是用一对J抠底
- **结果**：闲家上台（成为新庄家），但不降级，新庄家从J继续打起

**场景三**：打J，闲家得分35分，最后一轮用一对J抠底
- 条件① ✓ 级牌是J
- 条件② ✗ 35分 < 80分
- 条件③ ✓ 最后一轮用一对J抠底
- **结果**：闲家得分不足80分，庄家方升2级（小光），不触发降级

**场景四**：打J，闲家得分35分，最后一轮用一对J抠底（落实政策）
- 条件① ✓
- 条件② ✗（35<40，属于“落实政策”范围）
- 条件③ ✓
- **结果**：庄家保级，下一局继续打J

**场景五**：打A，闲家得分95分，最后一轮用一对A抠底
- 条件① ✓ 级牌是A
- 条件② ✓ 95分 ≥ 80分
- 条件③ ✓ 最后一轮用一对A抠底
- **结果**：庄家降级到J并下庄

**场景六**：打A，闲家得分95分，最后一轮用一对K抠底
- 条件① ✓ 级牌是A
- 条件② ✓ 95分 ≥ 80分
- 条件③ ✗ 最后一轮不是用一对A抠底
- **结果**：闲家上台（成为新庄家），从A继续打起

### 13.5 降级后的级牌变化图示
```
打J时降级：
J → 2

打A时降级：
A → J → ...（下一局从J开始，之后正常升级）
```

## 十四、必打牌限制与扣级规则

### 14.1 必打牌
**2、J、A** 是必打级牌，无论升级还是降级，都不能跳过这三张牌。

### 14.2 升级时的扣级规则
当一方升级时，如果计算出的升级结果要跳过必打牌，则**跳过的级数由对方（庄家方）倒退补齐**。

**扣级规则计算公式**：
1. 先计算闲家应升级数（按得分）
2. 从当前级牌开始，逐级向上数，遇到必打牌则必须停下
3. 闲家实际能升到的级数 = 当前级牌 + 可升的级数（直到遇到必打牌前）
4. 多出来的级数由庄家方倒退扣除

**示例1**：
- 庄家当前打**6**
- 闲家得分为140分（应升3级：6→7、8、9）
- 9之后是J（必打），不能跳过
- 闲家最终只能打到**9**（升2级）
- 少升的1级从庄家扣除，庄家从6**倒退1级**到5
- 结果：下一局庄家打5

**示例2**：
- 庄家当前打**Q**
- 闲家得分为160分（应升4级：Q→K、A、2、5）
- 但A是必打，不能跳过
- 闲家只能打到A（升2级：Q→K、A）
- 少升的2级从庄家扣除，庄家倒退2级（Q→10）
- 结果：下一局庄家打10

**示例3**：
- 庄家当前打**K**
- 闲家得分为180分（应升5级：K→A、2、5、10、J）
- A是必打，K→A升1级后必须停下
- 闲家实际只升1级到A
- 少升的4级从庄家扣除，庄家倒退4级（K→10？需具体计算）
- 但倒退不能低于2，所以庄家可能降到2

### 14.3 降级时的必打规则
降级时同样不能跳过必打牌：

**示例**：
- 庄家打A时被降级，本应降到J
- J是必打牌，符合规则
- 如果某种特殊情况要降到必打牌之后，必须停在必打牌上

### 14.4 必打牌意义总结
必打牌的存在确保了游戏的关键节点：
- **2**：起点，每次循环的开始
- **J**：第一个关口，被一对J抠底会直接打回2
- **A**：终点前的最后关口，被一对A抠底会倒退到J

## 十五、违规与处罚

### 15.1 不跟牌、出错牌
| 违规行为                             | 处罚           |
| ------------------------------------ | -------------- |
| 有首家花色但不跟（故意出其他花色）   | 每错一张罚10分 |
| 首家出对子，你有对子但不出对子       | 每错一张罚10分 |
| 首家出拖拉机，你有拖拉机但不出拖拉机 | 每错一张罚10分 |
| 出牌落地后反悔更换                   | 罚10分         |
| 多次故意违规                         | 可判本局负     |

### 15.2 多牌或少牌
- **庄家多牌或少牌**：倒庄（下台），对方（闲家）升2级
- **闲家多牌或少牌**：庄家方升2级
- 比赛开始后发现多牌少牌，无法补救时按此处理

### 15.3 扣错底牌
- **庄家扣错底牌**（如扣了8张但数量不对，或扣牌后手牌数量不对）：自动下台，对方升2级
- **如果当时正在打J/A**：按降级规则处理（降至J或2）
- **扣底时误将分牌以外的牌放入**：如果违反规则（如不允许扣分牌但扣了），按扣错底牌处理

### 15.4 甩牌失败
- 每错一张牌，罚10分
- 罚分可计入对方得分，或从本方总分扣除（赛前约定）
- 多次甩牌失败可累计处罚

### 15.5 打J/A时庄家方出错牌
- 如果庄家方在打J/A的关键局出现严重违规（如多牌少牌、扣错底牌）
- 可视为自动降级处理：打J时降至2，打A时降至J
- 具体处罚力度由赛前约定

### 15.6 争议处理
- 出现争议时，以裁判（或事先约定的规则）为准
- 无法判定时，通常重打该局
